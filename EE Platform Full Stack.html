Data Format Specification

Required Fields

Field Unit Description
project_id string Unique project identifier
timestamp ISO 8601 Reading time (YYYY-MM-DD HH:MM:SS)
flow_in m³/h Chilled water flow rate in
flow_out m³/h Chilled water flow rate out
temp_in °C Chilled water temperature in
temp_out °C Chilled water temperature out
power kW Chiller power consumption
ambient_temp °C Cooling tower ambient temperature

JSON Example

{ "project_id": "PROJ-001", "timestamp": "2026-03-12 13:45:00", "flow_in": 150.5, "flow_out": 150.2, "temp_in": 12.5, "temp_out": 7.0, "power": 85.3, "ambient_temp": 32.5 }

CSV Example

project_id,timestamp,flow_in,flow_out,temp_in,temp_out,power,ambient_temp PROJ-001,2026-03-12 13:45:00,150.5,150.2,12.5,7.0,85.3,32.5 PROJ-001,2026-03-12 14:00:00,152.0,151.8,12.3,7.2,86.1,32.8

Efficiency Calculation

The system automatically calculates cooling efficiency using:

Cooling Load (TR) = Flow × ΔT × 4.187 / 3.517
Where: ΔT = temp_in - temp_out

Efficiency (kW/TR) = Power / Cooling Load

Lower kW/TR values indicate better efficiency (typical range: 0.5-1.2 kW/TR)
